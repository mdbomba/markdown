# Templates commands

This directory documents the `4` LoadMaster REST primitives grouped under the **templates** category.

| Command | Description |
|---------|-------------|
| [`access/uploadtemplate`](access-uploadtemplate.md) | Installs template |
| [`access/deltemplate`](access-deltemplate.md) | Removes template |
| [`access/listtemplates`](access-listtemplates.md) | Retrieves template |
| [`access/uploadvserrfile`](access-uploadvserrfile.md) | Installs vserror file |
