# Diagnostics commands

This directory documents the `13` LoadMaster REST primitives grouped under the **diagnostics** category.

| Command | Description |
|---------|-------------|
| [`access/showhealth`](access-showhealth.md) | Retrieves adc service health |
| [`access/modhealth`](access-modhealth.md) | Updates adc service health |
| [`access/showadaptive`](access-showadaptive.md) | Retrieves adc adaptive health check |
| [`access/modadaptive`](access-modadaptive.md) | Updates adc adaptive health check |
| [`access/tcpdump`](access-tcpdump.md) | Handles TCP traffic |
| [`access/resolvenow`](access-resolvenow.md) | Updates network dnscache |
| [`access/opensslspeed`](access-opensslspeed.md) | Handles opensslspeed |
| [`access/isopensslspeedrunning`](access-isopensslspeedrunning.md) | Handles isopensslspeedrunning |
| [`access/smartctl`](access-smartctl.md) | Handles smartctl |
| [`access/getraiddisksinfo`](access-getraiddisksinfo.md) | Retrieves lm RAID controller disk |
| [`access/getraidinfo`](access-getraidinfo.md) | Retrieves lm RAID controller |
| [`access/showtelemetry`](access-showtelemetry.md) | Retrieves telemetry |
| [`access/enabletelemetry`](access-enabletelemetry.md) | Enables telemetry |
| [`access/listifconfig`](../network/access-listifconfig.md) | Shared network interface listing primitive |
