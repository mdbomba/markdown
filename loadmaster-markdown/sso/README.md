# SSO commands

This directory documents the `10` LoadMaster REST primitives grouped under the **SSO** category.

| Command | Description |
|---------|-------------|
| [`access/setrsaconfig`](access-setrsaconfig.md) | Installs ssorsaconfiguration file |
| [`access/setrsanodesecret`](access-setrsanodesecret.md) | Installs ssorsanode secret and password |
| [`access/uploadsamlidpmd`](access-uploadsamlidpmd.md) | Is used to upload an IdP metadata file |
| [`access/downloadsamlspcert`](access-downloadsamlspcert.md) | Exports samlspcert |
| [`access/ssoimages`](access-ssoimages.md) | Handles ssoimages |
| [`access/delssoimage`](access-delssoimage.md) | Deletes ssoimage |
| [`access/listssoimages`](access-listssoimages.md) | Lists ssoimages |
| [`access/ssodomain`](access-ssodomain.md) | Handles ssodomain |
| [`access/showdomainlockedusers`](access-showdomainlockedusers.md) | Lists currently locked users for the specified SSO domain |
| [`access/unlockdomainusers`](access-unlockdomainusers.md) | Unlocks one or more locked users for the specified SSO domain |
