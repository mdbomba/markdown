# Network commands

This directory documents the `20` LoadMaster REST primitives grouped under the **network** category.

| Command | Description |
|---------|-------------|
| [`access/showiface`](access-showiface.md) | Retrieves configuration details for a network interface |
| [`access/modiface`](access-modiface.md) | Modifies a network interface configuration |
| [`access/addadditional`](access-addadditional.md) | Adds additional |
| [`access/deladditional`](access-deladditional.md) | Deletes additional |
| [`access/createbond`](access-createbond.md) | Registers network bonded interface |
| [`access/unbond`](access-unbond.md) | Handles network bonded interface |
| [`access/addbond`](access-addbond.md) | Creates network bonded interface |
| [`access/delbond`](access-delbond.md) | Removes network bonded interface |
| [`access/addvlan`](access-addvlan.md) | Adds VLAN |
| [`access/delvlan`](access-delvlan.md) | Deletes VLAN |
| [`access/addvxlan`](access-addvxlan.md) | Adds VXLAN |
| [`access/delvxlan`](access-delvxlan.md) | Deletes VXLAN |
| [`access/listifconfig`](access-listifconfig.md) | Lists configured network interfaces and addressing information |
| [`access/addroute`](access-addroute.md) | Adds a static route to the appliance routing table |
| [`access/delroute`](access-delroute.md) | Deletes a static route from the appliance routing table |
| [`access/showroute`](access-showroute.md) | Retrieves the configured static routes |
| [`access/addnocache`](access-addnocache.md) | Adds no cache |
| [`access/delnocache`](access-delnocache.md) | Deletes no cache |
| [`access/addnocompress`](access-addnocompress.md) | Adds no compression |
| [`access/delnocompress`](access-delnocompress.md) | Deletes no compression |
