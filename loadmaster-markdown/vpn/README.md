# VPN commands

This directory documents the `17` LoadMaster REST primitives grouped under the **VPN** category.

| Command | Description |
|---------|-------------|
| [`access/createvpncon`](access-createvpncon.md) | Creates lm VPN connection |
| [`access/deletevpncon`](access-deletevpncon.md) | Removes lm VPN connection |
| [`access/listvpns`](access-listvpns.md) | Lists vpns |
| [`access/getvpnstatus`](access-getvpnstatus.md) | Retrieves vpnstatus |
| [`access/setvpnaddr`](access-setvpnaddr.md) | Updates lm VPN addrs |
| [`access/setvpnlocalip`](access-setvpnlocalip.md) | Updates lm VPN local IP |
| [`access/setvpnlocalsubnets`](access-setvpnlocalsubnets.md) | Updates lm VPN local subnet |
| [`access/setvpnremoteip`](access-setvpnremoteip.md) | Updates lm VPN remote IP |
| [`access/setvpnremotesubnets`](access-setvpnremotesubnets.md) | Updates lm VPN remote subnet |
| [`access/setvpnsecret`](access-setvpnsecret.md) | Updates lm VPN secret |
| [`access/setvpnpfsdisable`](access-setvpnpfsdisable.md) | Updates lm VPN pfs disable |
| [`access/setvpnpfsenable`](access-setvpnpfsenable.md) | Updates lm VPN pfs enable |
| [`access/startvpncon`](access-startvpncon.md) | Starts lm VPN connection |
| [`access/stopvpncon`](access-stopvpncon.md) | Stops lm VPN connection |
| [`access/startikedaemon`](access-startikedaemon.md) | Starts lm VPN ike daemon |
| [`access/stopikedaemon`](access-stopikedaemon.md) | Stops lm VPN ike daemon |
| [`access/statusikedaemon`](access-statusikedaemon.md) | Retrieves lm VPN ike daemon status |
