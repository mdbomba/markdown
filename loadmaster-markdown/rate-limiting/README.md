# Rate Limiting commands

This directory documents the `20` LoadMaster REST primitives grouped under the **rate limiting** category.

| Command | Description |
|---------|-------------|
| [`access/addlimitrule`](access-addlimitrule.md) | Creates adc limit rule |
| [`access/dellimitrule`](access-dellimitrule.md) | Removes adc limit rule |
| [`access/modlimitrule`](access-modlimitrule.md) | Updates adc limit rule |
| [`access/movelimitrule`](access-movelimitrule.md) | Handles adc limit rule |
| [`access/listlimitrules`](access-listlimitrules.md) | Retrieves adc limit rules |
| [`access/afeclientlimitadd`](access-afeclientlimitadd.md) | Creates lm IP connection limit |
| [`access/afeclientlimitdel`](access-afeclientlimitdel.md) | Removes lm IP connection limit |
| [`access/afeclientlimitlist`](access-afeclientlimitlist.md) | Retrieves lm IP connection limit |
| [`access/clientbandwidthlimitadd`](access-clientbandwidthlimitadd.md) | Creates client bandwidth limit |
| [`access/clientbandwidthlimitdel`](access-clientbandwidthlimitdel.md) | Removes client bandwidth limit |
| [`access/clientbandwidthlimitlist`](access-clientbandwidthlimitlist.md) | Retrieves client bandwidth limit |
| [`access/clientcpslimitadd`](access-clientcpslimitadd.md) | Creates client cpslimit |
| [`access/clientcpslimitdel`](access-clientcpslimitdel.md) | Removes client cpslimit |
| [`access/clientcpslimitlist`](access-clientcpslimitlist.md) | Retrieves client cpslimit |
| [`access/clientmaxclimitadd`](access-clientmaxclimitadd.md) | Creates client maxc limit |
| [`access/clientmaxclimitdel`](access-clientmaxclimitdel.md) | Removes client maxc limit |
| [`access/clientmaxclimitlist`](access-clientmaxclimitlist.md) | Retrieves client maxc limit |
| [`access/clientrpslimitadd`](access-clientrpslimitadd.md) | Creates client rpslimit |
| [`access/clientrpslimitdel`](access-clientrpslimitdel.md) | Removes client rpslimit |
| [`access/clientrpslimitlist`](access-clientrpslimitlist.md) | Retrieves client rpslimit |
