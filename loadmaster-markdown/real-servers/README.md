# Real Servers commands

This directory documents the `8` LoadMaster REST primitives grouped under the **real servers** category.

| Command | Description |
|---------|-------------|
| [`access/enablers`](access-enablers.md) | Enables a real server that is currently disabled or draining |
| [`access/disablers`](access-disablers.md) | Disables a real server on a virtual service |
| [`access/addrs`](access-addrs.md) | Adds a real server to a virtual service |
| [`access/delrs`](access-delrs.md) | Deletes a real server from a virtual service |
| [`access/modrs`](access-modrs.md) | Modifies the settings of a real server |
| [`access/showrs`](access-showrs.md) | Retrieves details for one real server attached to a virtual service |
| [`access/addrsrule`](access-addrsrule.md) | Associates a rule with a real server |
| [`access/delrsrule`](access-delrsrule.md) | Removes a rule association from a real server |
