# Licensing Extra commands

This directory documents the `9` LoadMaster REST primitives grouped under the **licensing extra** category.

| Command | Description |
|---------|-------------|
| [`access/license`](access-license.md) | Uploads an offline license request or response file to the appliance |
| [`access/spla_license`](access-spla_license.md) | Runs the SPLA licensing workflow against Progress licensing services |
| [`access/kill_spla_instance`](access-kill_spla_instance.md) | Removes a SPLA instance registration from the licensing service |
| [`access/kill_instance`](access-kill_instance.md) | Removes a named instance registration from the licensing service |
| [`access/licenseinfo`](access-licenseinfo.md) | Returns current appliance license details |
| [`access/killaslinstance`](access-killaslinstance.md) | Stops the currently associated on-premise ASL licensing instance |
| [`access/getaslinfo`](access-getaslinfo.md) | Retrieves on-premise ASL licensing server details |
| [`access/aslactivate`](access-aslactivate.md) | Activates the appliance against an on-premise ASL licensing server |
| [`access/aslgetlicensetypes`](access-aslgetlicensetypes.md) | Lists available license types from an on-premise ASL server |
