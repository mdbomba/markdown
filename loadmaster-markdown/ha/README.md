# HA commands

This directory documents the `13` LoadMaster REST primitives grouped under the **HA** category.

| Command | Description |
|---------|-------------|
| [`access/cluster`](access-cluster.md) | Handles cluster |
| [`access/setlmcommsecret`](access-setlmcommsecret.md) | Updates lmcomm secret |
| [`access/awshamode`](access-awshamode.md) | Updates lm AWS hamode |
| [`access/awshaparam`](access-awshaparam.md) | Updates lm AWS haconfiguration |
| [`access/azurehamode`](access-azurehamode.md) | Updates lm azure hamode |
| [`access/azurehaparam`](access-azurehaparam.md) | Updates lm azure haconfiguration |
| [`access/getCloudHaParams`](access-getCloudHaParams.md) | Retrieves cloud HA params |
| [`access/getawshaparams`](access-getawshaparams.md) | Retrieves lm AWS haconfiguration |
| [`access/getazurehaparams`](access-getazurehaparams.md) | Retrieves lm azure haconfiguration |
| [`access/getcloudhaparameters`](access-getcloudhaparameters.md) | Retrieves cloudhaparameters |
| [`access/getcloudhaparams`](access-getcloudhaparams.md) | Retrieves lm cloud HA configuration |
| [`access/setcloudhamode`](access-setcloudhamode.md) | Updates lm cloud hamode |
| [`access/setcloudhaparam`](access-setcloudhaparam.md) | Updates lm cloud HA configuration |
