# Hosts commands

This directory documents the `3` LoadMaster REST primitives grouped under the **hosts** category.

| Command | Description |
|---------|-------------|
| [`access/gethosts`](access-gethosts.md) | Retrieves hosts entry |
| [`access/addhostsentry`](access-addhostsentry.md) | Creates hosts entry |
| [`access/delhostsentry`](access-delhostsentry.md) | Removes hosts entry |
