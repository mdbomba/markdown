# Rules commands

This directory documents the `12` LoadMaster REST primitives grouped under the **rules** category.

| Command | Description |
|---------|-------------|
| [`access/showrule`](access-showrule.md) | With no parameters will list all of the existing rules |
| [`access/addrule`](access-addrule.md) | Creates adc content rule |
| [`access/delrule`](access-delrule.md) | Removes adc content rule |
| [`access/modrule`](access-modrule.md) | Updates adc content rule |
| [`access/addprerule`](access-addprerule.md) | Adds prerule |
| [`access/delprerule`](access-delprerule.md) | Deletes prerule |
| [`access/addrequestrule`](access-addrequestrule.md) | Adds requestrule |
| [`access/delrequestrule`](access-delrequestrule.md) | Deletes requestrule |
| [`access/addresponserule`](access-addresponserule.md) | Adds responserule |
| [`access/delresponserule`](access-delresponserule.md) | Deletes responserule |
| [`access/addresponsebodyrule`](access-addresponsebodyrule.md) | Creates adc virtual service response body rule |
| [`access/delresponsebodyrule`](access-delresponsebodyrule.md) | Removes adc virtual service response body rule |
