# LDAP commands

This directory documents the `6` LoadMaster REST primitives grouped under the **LDAP** category.

| Command | Description |
|---------|-------------|
| [`access/addldapendpoint`](access-addldapendpoint.md) | Creates LDAP endpoint |
| [`access/deleteldapendpoint`](access-deleteldapendpoint.md) | Removes LDAP endpoint |
| [`access/modifyldapendpoint`](access-modifyldapendpoint.md) | Updates LDAP endpoint |
| [`access/showldapendpoint`](access-showldapendpoint.md) | Shows ldapendpoint |
| [`access/showldaplist`](access-showldaplist.md) | Shows ldaplist |
| [`access/aclcontrol`](access-aclcontrol.md) | Removes global packet filter ACL |
